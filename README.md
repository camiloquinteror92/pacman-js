# 👾 Pac-Man (JavaScript)

A browser-based Pac-Man clone built with vanilla JavaScript, HTML5 and CSS — no frameworks, no build step.

## Features
- Classic maze, pellets and ghost movement
- Keyboard controls (arrow keys)
- Pure client-side: runs by just opening the page

## Run locally
Clone the repo and open `index.html` in any browser.

## Tech
`HTML5` · `CSS3` · `JavaScript`
